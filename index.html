Allgemein

Als die wohl bekannteste deutschsprachige BSD-Community haben wir - neben dem Forum selbst – das Wiki. Die allgemeingültigen Ergebnisse unseres Erfahrungsaustausches sollen (konsolidiert) auffindbar und letztlich Hilfe sein.

Es gibt eine unzählige Anzahl von Themen. Wir pflegen Inhalte zu den verschieden – aber eben auch ähnlichen – Arten von Betriebssystemen, die sich um BSD (Unices) entwickelt haben. Zentral geht um die Nutzung und Verwendung, entsprechend den verschiedenen Fällen an Anwendungen. Die Ausführungen sollen über die Handbücher – und andere direkt bereitgestellten Dokumentationen – hinausgehen. Es ist die größte deutschsprachige Sammlung derartiger Inhalte, die frei (CC-Lizenzierung) zur Verfügung stehen.

Finden von Inhalten

Neben der einfachen textlichen Suche, verwenden wir zum Finden von Inhalten Tags. Tags bezeichnen Marker für Elemente in einem Dokument.

Verbessern der Inhalte

Wir verwenden Sphinx, welches reStructuredText oder Markdown in HTML-Seiten sowie in vielen weiteren Formaten, wie PDF, EPUB oder Manpages ausgeben kann.

Neue Inhalte können einfach via Git und Pull Requests in das Repository auf Github.com eingepflegt werden!

Eine kleine Anleitung für Git gibt es hier.
